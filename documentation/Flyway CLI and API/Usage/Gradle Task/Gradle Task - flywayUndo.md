# Gradle Task: flywayUndo
{% include teams.html %}

[Undoes](Commands/undo) the most recently applied versioned migration.

<a href="Commands/undo"><img src="assets/command-undo.png" alt="undo"></a>

## Usage
<pre class="console">&gt; gradle flywayUndo</pre>

## Configuration
See [configuration](Configuration/parameters) for a full list of supported configuration parameters.
