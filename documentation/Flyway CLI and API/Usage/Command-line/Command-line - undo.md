# Command-line: undo
{% include teams.html %}

[Undoes](Commands/undo) the most recently applied versioned migration.

<a href="Commands/undo"><img src="assets/command-undo.png" alt="undo"></a>

## Usage

<pre class="console"><span>&gt;</span> flyway [options] undo</pre>

## Options

See [configuration](Configuration/parameters) for a full list of supported configuration parameters.

```properties

flyway.schemas=schema1,schema2,schema3
flyway.table=schema_history

flyway.sqlMigrationPrefix=Migration-
flyway.undoSqlMigrationPrefix=downgrade
flyway.repeatableSqlMigrationPrefix=RRR
flyway.sqlMigrationSeparator=__
flyway.sqlMigrationSuffixes=.sql,.pkg,.pkb
```
